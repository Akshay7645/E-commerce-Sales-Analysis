#  Project Overview

This project focuses on analyzing Ecommerce Sales Data to uncover meaningful business insights using data analysis and visualization techniques.

The analysis includes:

* Monthly Sales Analysis  
* Profit Analysis  
* Category Performance  
* Customer Segment Analysis  
* Sales-to-Profit Ratio  
* Interactive Visualizations






##  Business Questions Solved

This project answers several key business questions:

* Which month generates the highest sales?
* Which month generates the highest profit?
* Which product category contributes the most revenue?
* Which sub-categories perform best and worst?
* Which customer segment generates the highest sales and profit?
* What is the sales-to-profit ratio across customer segments?
* Which areas of the business are generating losses?
* How can sales performance be improved using data-driven insights?
