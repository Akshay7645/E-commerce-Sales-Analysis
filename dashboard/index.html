📊 Ecommerce Sales Analysis

Interactive Data Analysis Dashboard using Python & Plotly

Project Overview

This project analyzes ecommerce sales data to identify monthly sales trends, category performance, profit analysis, and customer segment insights.

Tools Used

  • Python
  • Pandas
  • Plotly
  • Jupyter Notebook
  • HTML & CSS

Key Insights

  • Technology category generated highest sales.
  • December showed highest monthly profit.
  • Consumer segment had highest sales contribution.
  • Some sub-categories generated negative profit.
